# tic-tac-toe

A simple 2-player Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript.

## Features

- Classic 3x3 grid gameplay
- Two-player mode (X vs O)
- Win and draw detection
- Clean, responsive UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla, no frameworks/libraries)

## How to Run

1. Clone the repository
```bash
   git clone https://github.com/Aishwarya07042005/tic-tac-toe.git
```
2. Open `index.html` in your browser — no build steps or dependencies needed.

## How to Play

- Players take turns clicking on an empty cell to place their mark (X or O)
- First player to get 3 marks in a row (horizontally, vertically, or diagonally) wins
- If all 9 cells fill up with no winner, the game ends in a draw
